# Mama's Crafts - site_mom

This repository will contain the marketplace for moms to sell handmade crafts. This commit adds a simple static prototype (HTML + Tailwind CSS) for the landing page so we have a visual starting point.

Planned next steps:
- Replace static prototype with Next.js + Tailwind scaffold on branch feature/marketplace-mvp
- Integrate Supabase for auth/storage and Stripe for payments
- Implement seller dashboard and product CRUD

How to view locally:
1. Open index.html in the browser (no build required for this prototype).
2. For development, consider installing a static server (e.g., npm i -g serve) and run `serve .`.

Branching:
- This initial commit is on the `main` branch. I will create `feature/marketplace-mvp` next and push the full app scaffold.
